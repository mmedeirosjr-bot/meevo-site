# MEEVO V6

Ajustes feitos:
- hospitais salvos agora ficam em seletor, sem poluir a tela
- exame físico dividido em AN, AR, ACV, ABD e EXT
- modelos de exame físico com preenchimento rápido
- evoluções salvas podem ser selecionadas e abertas para edição
- exportação em PDF agora usa a evolução salva/selecionada, não apenas o texto atual dos campos
- prévia da evolução salva antes de exportar

Como publicar:
1. Extraia o ZIP.
2. Entre em https://app.netlify.com/drop
3. Arraste a pasta extraída.
4. Use o novo link gerado.
